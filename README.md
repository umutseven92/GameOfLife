# Game Of Life
Simple Conway's Game of Life, written in Python.

To run `cd` into `/src`, and do: 
```python
python cgl.py
```

You need to have pygame installed.

## Controls
* `R` key to reset
* `P` key to pause
* Right arrow to speed up
* Left arrow to slow down
* `Esc` to pause
